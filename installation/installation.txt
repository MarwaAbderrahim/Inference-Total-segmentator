####### Installation 

conda install pytorch==2.0.0 torchvision==0.15.0 torchaudio==2.0.0 pytorch-cuda=11.8 -c pytorch -c nvidia

pip install totalsegmentator

pip install -r requirements-dev.txt


####### Laste step 

After preparing the environment and installing the requirements, you need to replace all the scripts located at:

environement/lib/site-packages/totalsegmentator

with the scripts found in the Git repository:

/totalsegmentator
